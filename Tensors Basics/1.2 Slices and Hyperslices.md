### 1.2 - Slices and Hyperslices

> **Definition (Slices of 3-way Tensor)**  
> Let $\mathbf{X}$ be a 3-way tensor of size $m \times n \times p$.  
> The $i$-th **horizontal slice** is a matrix of size $n \times p$ given by $\mathbf{X}(i,:,:)$.  
> The $j$-th **lateral slice** is a matrix of size $m \times p$ given by $\mathbf{X}(:,j,:)$.  
> The $k$-th **frontal slice** is a matrix of size $m \times n$ given by $\mathbf{X}(:,:,k)$.
<br>

```text
  +--------------+        +--------------+          +--------------+
 /              /|       /  /  /  /  /  /|         /--------------/|
+-------------+ /|      +-------------+  |        +-------------+ ||
|_____________|//|      | |  |  |  |  |  |        |             | ||
|_____________|//|      | |  |  |  |  |  |        |             | ||
|_____________|//+      | |  |  |  |  |  +        |             | |+
|_____________|//       | |  |  |  |  | /         |             |/
+-------------+         +-------------+           +-------------+
a) Horizontal slices, X(i,:,:)   b) Lateral slices, X(:,j,:)   c) Frontal slices, X(:,:,k)
```
<br>


>**Definition (Frontal Slices of a *d*-way Tensor** <br>
> The frontal slices of a *d*-way tensor $\textbf{X}$ of size $n_1 \times n_2 \times \dots \times n_d$ are given by $\textbf{X}(:, :, i_3, i_4,\dots , i_d)$ for all $(i_3, i_4, \dots , i_d)$ in $[n_3] \times [n_4] \times \dots \times [n_d]$.

<br>

We define a **hyperslice** to be a subtensor defined by fixing a single index, which is called a **mode-*k* hyeprslice**.
So in a third-order tensor, mode-1 hyperslices are called horizontal, mode-2 hyperslices are called lateral, mode-3 hyperslices are called frontal. *(Only named for a 3-way case)*

<br>

>**Definition (Mode-*k* Hyperslice)** <br>
>The mode-*k* hyperslice of a *d*-way tensor $\textbf{X}$ of size $n_1 \times n_2 \times \dots \times n_d$ is a *(d-1)*-way tensor of size $n_1 \times \dots \times n_{k-1} \times n_{k+1} \times \dots \times n_d$. <br>
>The $j_{th}$ mode-*k* hyperslice is given by $\textbf{X}(:, \dots ,: ,j , :, \dots , :)$.
